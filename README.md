# Personal Link Hub

This project is a simple personal landing page that showcases a profile, a short intro, and a set of clickable links to social platforms and other online spaces.

## What it does
- Displays a profile image and name
- Shows a short description about the person
- Provides quick-access buttons for links such as Portfolio, YouTube, GitHub, and Spotify
- Includes a dark/light mode toggle for a better viewing experience

## What it is meant for
This page is meant to act as a compact personal link hub or online profile page. It is useful for sharing a single place where friends, followers, or employers can quickly find important links and learn a little about the person behind the page.

## Files
- `index.html` — The main page structure
- `style.css` — The styling and theme behavior

## How to use
Open `index.html` in a browser to view the page.
